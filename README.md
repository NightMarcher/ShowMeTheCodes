# ShowMeTheCodes
Little interesting scripts with Python3, for recording and learning. 

## Python Scripts
### 1.activationCode.py
生成指定数量，指定长度的激活码，并判断输入的激活码是否有效（在生成的激活码列表中即为有效）
### 2.addNum2Avatar.py
向头像上添加红色数字
### 3.cardSecurityCode.py
随机生成包含四个大写字母的验证码
### 4.drawFigure.py
绘制图表示例
### 5.image2Ascii.py
图像转字符画
